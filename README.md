## 📖 About The Project

**ChatrFinance** is a comprehensive real-time financial dashboard for the Iranian market and global exchanges.

This mobile application provides instant access to:

- 💰 **Gold & Currency** — Live rates in Iranian Rial (IRR)
- 📈 **Tehran Stock Exchange** — Top stocks and indices
- 🪙 **Cryptocurrencies** — Bitcoin, Ethereum, and more
- 🌍 **Global Stocks** — NVIDIA, Apple, Tesla, etc.
- 🛢️ **Commodities** — Crude oil, gold spot, and more
- 💱 **Forex** — USD, EUR, GBP, AED, and other currencies to IRR
- 📊 **TradingView Charts** — Professional charting integration
- 🗞️ **Persian News Feed** — Curated financial news in Farsi
- ✅ **Verified Analysts** — Expert insights with admin verification
